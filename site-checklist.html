デザイン確認

スマホ表示とデザインルール

赤×白を基調に、ゴールドはアクセントとして使用します。本文・比較表・カードは白背景で可読性を優先します。

スマホ表示の確認

  • スマホ時にヘッダーが詰まりすぎていないか
  • ヒーローの余白が大きすぎないか
  • キャラクター画像が大きすぎないか
  • カードが1列で読みやすく表示されているか
  • 比較表が横スクロールできるか

デザイン方針の確認

  • 本文背景が白中心で読みやすいか
  • 赤とゴールドの使い方が派手すぎないか
  • 炎や熱量の演出を使いすぎていないか
  • CTAが比較・確認寄りの文言になっているか
  • フッターリンクがスマホでも崩れていないか
表現ルール

禁止表現と推奨表現の確認

勝利保証や損失回収を連想させる表現は避け、比較・確認・慎重な判断を促す表現に統一します。

使わない表現

射幸心を強く煽る表現、保証表現、損失回収を連想させる表現は使用しません。

  • 勝てる、必ず勝てる
  • 稼げる、儲かる
  • 取り返せる、損失回収
  • 勝利保証、出金保証、賞金獲得保証
  • 今すぐ登録しないと損
  • 登録必須、急いで参加

使ってよい表現

条件確認、公式規約確認、慎重な比較を促す表現を使います。

  • 条件を確認する
  • 公式規約を確認する
  • 賭け条件を理解する
  • 最大出金を確認する
  • 対象ゲームを確認する
  • 不安がある場合は利用を急がない

TOP、レビュー一覧、カジノ一覧、ボーナス比較、入出金ガイド、初心者ガイドを相互に行き来できる状態にします。

責任ある利用

全ページ共通で入れたい注意文

各ページでは、利用を推奨・保証するものではないこと、公式規約と対象地域を確認すること、不安がある場合は利用を急がないことを明記します。

責任ある利用ブロックの確認

ボーナス比較、入出金ガイド、初心者ガイドなどでは、勝利保証や出金保証ではなく、 条件確認と慎重な判断を促す文脈に統一します。 予算や時間を決められない場合、不安がある場合は利用を急がないという案内を入れます。

公開前の最終チェック

GitHub Pagesで公開後、スマホとPCの両方で表示を確認します。 特に、画像ファイル名の一致、ロゴ画像の表示、比較表の横スクロール、フッターリンク、レビュー10本へのリンク、 誇大表現の有無、責任ある利用ブロックの有無を確認してください。

  • 全ページのヘッダーリンクが正しく開く
  • 全ページのフッターリンクが正しく開く
  • 比較表がスマホで横スクロールできる
  • マスコット画像が大きすぎず自然に表示される
  • 既存ロゴ画像が消えていない
  • 勝利保証・損失回収・出金保証の表現がない
  • ボーナス条件確認の文脈になっている
  • 本人確認と出金条件への導線がある
  • 責任ある利用の注意文がある
  • SEO title と description がページ内容に合っている
